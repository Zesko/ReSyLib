    Welcome to the Reactive Systems Beaglebone Black C++11 library project by the "deeply embedded" lecture class of Stephan Pareigis.

    We develop a C++11 library for embedded reactive systems. The library is testet on a beaglebone black.

    Goal of the Library

    The goal of the BBBlib is to provide easy GPIO-access for the beaglebone black. Additionally samples of C++11 code are provided which are commonly used in reactive systems: hardware abstraction layer, proxy, finite state machine, active object, and others

    Contact

    This project is part of a lecture class by Prof Stephan Pareigis at Hamburg University of Applied Sciences, Department Computer Science. Feel free to contact me for information at stephan.pareigis@haw-hamburg.de
